# Knights Challenge 
<br/>
<br/>
O desafio Knights visa a construcao de um sistema cadastro de herois utilizando as seguintes tecnologias:
<br/>

## Tecnologias 
    - Vue.js
    - NodJS (Express.js | NestJS) | C# (.NET 7+)
    - MongoDB | Redis
    - Servicos em containers (script para build e geracao)
